Global Disaster Impact Analysis

About this visualization

What does this diagram show? This scatterplot displays disaster events from around the world, visualizing the relationship between time (x-axis) and the total number of deaths (y-axis) for each event.

Color coding:

  • Green = Natural disasters (earthquakes, floods, storms, etc.)
  • Red = Technological disasters (accidents, industrial incidents, transport accidents, etc.)

Interactive features: Use the dropdown menu below to filter the visualization by country. Hover over any data point to see detailed information about the specific event (country, year, total deaths, disaster group, subgroup, type, and subtype).
